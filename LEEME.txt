- Abran phpmyadmin y borren la base de datos con el nombre "bd_sofit" si la tienen. Luego vuelven a 
crear otra base de datos con el nombre "bd_sofit" y cotejamiento "utf8_general_ci"

- Peguen cualquiera de estos link: http://localhost/SofitWeb_1.3/index.php ó  http://localhost:8080/SofitWeb_1.3/index.php
para los que tiene configurado apache con este servidor y ejecuten la aplicación.
